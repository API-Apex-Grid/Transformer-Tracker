Transformer Tracker API Spring Boot

Backend is up. Use the frontend UI to interact with the system, or call these API endpoints directly.

Frontend

  • Live app: apex-grid-transformer-tracker.vercel.app

API Endpoints

  • POST /api/login — obtain a token (provide username/password).
  • GET /api/transformers — list transformers.
  • GET /api/inspections — list inspections.
  • POST /api/inspections/{id}/upload — upload thermal image for an inspection.

Tip: Most endpoints are used by the frontend. Inspect network calls in the browser DevTools for details.

Local development

  • Run locally using the local branch. See the README for steps.
  • Default API base: http://localhost:8080
  • H2 Console (if enabled): /h2-console
